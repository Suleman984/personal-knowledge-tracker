# Personal Knowledge Tracker

A full-stack knowledge management platform that helps users capture, summarize, and retain information using AI-powered summarization and spaced repetition techniques.

**Tech Stack:** Go (Gin) • Next.js 15 • TypeScript • PostgreSQL • Redis • Chrome Extension • Google Gemini API

**Key Features:**
- AI-powered content summarization from URLs or text
- Chrome extension for quick content capture
- Automated spaced repetition email reminders
- Content management dashboard with search and categorization
